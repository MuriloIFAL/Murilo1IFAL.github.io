Seção 3 — Glossário Técnico

HTML
Linguagem de marcação para web.
Estrutura básica de um site.
JavaScript
Linguagem de programação para navegadores.
Permite criar conteúdo dinâmico.
HTTP
HTTPS
Protocolos de comunicação da web.
URL
Endereço Web
Identificador de recursos na rede.
DOM
Representação em árvore do documento HTML.
API
Interface para comunicação entre sistemas.
DNS
Resolução de nomes para IPs.
CSS
Folha de estilo para design.
